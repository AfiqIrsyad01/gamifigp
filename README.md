# 🎮 GamifiGP: PC Computer Store with Gamification

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen)](https://afiqirsyad01.github.io/gamifigp/)
[![Tech](https://img.shields.io/badge/backend-PHP%20%2F%20MySQL-blue)](#)

GamifiGP is a specialized e-commerce platform for PC components that integrates **gamification elements** to enhance user engagement. Unlike traditional stores, users are rewarded for their shopping journey.

## ✨ Key Features
- **🎮 Interactive Shopping:** Browse PC parts with a gaming-inspired UI.
- **🏆 Reward System:** Earn points/XP for purchases (Logic handled via PHP).
- **🛒 Dynamic Cart:** Real-time calculation of totals and potential rewards.
- **📱 Responsive Design:** Fully optimized for mobile and desktop gaming enthusiasts.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Animations), JavaScript (ES6)
- **Backend:** PHP (Private) - Handles session management & reward logic.
- **Database:** MySQL (Private) - Stores inventory and user "Level/XP" data.
